# neuro_cnc_helper
Веб-приложение для диагностики неисправностей станков с использованием нейросети.

## Установка и запуск

### 1. Скачайте проект
Скачайте архив с GitHub и распакуйте его в папку ваших проектов.

### 2. Установите библиотеки
Откройте **терминал** в корневой папке проекта и выполните команды:
```bash
pip install streamlit
pip install joblib
pip install numpy
pip install pandas
pip install scikit-learn
```
Если возникли проблемы с установкой или работой библиотек (приложения), создайте пустой проект и перенесите вручную все файлы.

### 3. Запустите приложение
В том же терминале выполните:

streamlit run app.py

### "Требования" 
(Python 3.9+)
